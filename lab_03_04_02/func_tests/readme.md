# Тесты для лабораторной работы №3 (задание lab_03_04_02)

## Входные данные
int a[N][M] - квадратная матрица из N (N<=10) строк и из M (M<=10) столбцов (N=M)

## Выходные данные
int number - найденное максимальное число, оканчивающееся на 5, находящееся под
побочной диагональю матрицы

# Позитивные тесты: 
- 01 - обычный тест (присутствуют разные элементы, удовлетворяющие условию)
- 02 - обычный тест (элементы, удовлетворяющие условию, одинаковы)
- 03 - под побочной диагональю есть отрицательное число, оканчивающееся на 5

# Негативные тесты:
- 01 - Ошибка ввода (ничего не введено)
- 02 - Ошибка ввода (отрицательное n)
- 03 - Ошибка ввода (нулевое n)
- 04 - Ошибка ввода (отрицательное m)
- 05 - Ошибка ввода (нулевое m)
- 06 - Ошибка ввода (буква вместо цифры в n)
- 07 - Ошибка ввода (буква вместо цифры в m)
- 08 - Ошибка ввода (буква вместо цифры в элементе матрицы)
- 09 - Ошибка ввода (введено недостаточное кол-во элементов)
- 10 - Введена не квадратная матрица
- 11 - В матрице нет элементов, оканчивающихся на 5 под побочной диагональю
